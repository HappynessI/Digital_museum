## 🚀 快速上手

1.打开终端，切换到frontend文件夹
```
cd ./frontend
```

2.安装前端依赖
```
npm install
```

3.启动前端服务
```
npm run serve
```

4.切换到backend文件夹
```
cd ./backend
```

5.安装后端依赖
```
npm install
```

6.启动后端服务
```
npm run dev
```

然后打开localhost链接即可
